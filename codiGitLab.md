# GitLab

## ¿Qué es GitLab?

GitLab es una plataforma web que permite gestionar repositorios Git y facilita el desarrollo colaborativo de software.

## Características principales

- Control de versiones con Git.
- Gestión de proyectos y tareas.
- CI/CD integrado
- Autoalojamiento(Self-hosting)

## ¿Para qué sirve?

Depende de para quien lo quiera usar:

- Equipos de desarrollo: Para gestionar código, colaborar y automatizar flujos de trabajo.

- Empresas: Para mejorar la eficiencia, seguridad y entrga de software a escala.

- Projectos personales: Tiene una version gratuita para individuos y proyectos pequeños.
