# Pull Request

## ¿Qué es una Pull Request?
Una Pull Request (PR) es una solicitud formal para integrar los cambios en un código desde una rama a otra, generalmente la rama principal.

## ¿Para qué sirve?
Sirve principalmente como un canal para que los desarrolladores expliquen su trabajo al equipo

## Ventajas
- Calidad del codigo
- Colaboracion y conocimiento.
- Estabbilidad del proyecto
- Visibilidad

## Desventajas
- Sobrecarga
- Requiere disciplina
- Dependencia
